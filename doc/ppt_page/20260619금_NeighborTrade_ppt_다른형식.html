Project Overview

프로젝트 목표와 범위

단순 상품 게시판이 아니라, 실제 동네 정보를 기반으로 거래 신뢰도를 높이는 하이퍼로컬 중고거래 서비스를 목표로 했습니다.

동네 기반 거래사용자 위치와 선택 동네를 검증하여 지역 기반 거래 환경 제공
거래 신뢰도동네 인증, 매너점수, 후기, 신고 기능을 통해 신뢰 보완
실시간 소통상품별 1:1 채팅방에서 WebSocket/STOMP 기반 메시지 송수신
거래 흐름거래 요청, 예약, 완료, 취소, 후기 작성까지 상태 흐름 구성
동네생활거래 외 지역 커뮤니티 기능과 댓글 흐름 제공
운영 관리관리자 대시보드, 신고 처리, 회원/상품/검색 관리 기능 구현

Issues & Lessons

  • MVP 범위와 확장 기능 분리
  • Playwright E2E와 통합 테스트 확대
  • 운영 로그, 모니터링, 장애 대응 체계 도입
  • 코드/SQL/보안/UI 리뷰 체계 보완

종합 결론

NeighborTrade는 동네 인증 기반 중고거래 MVP로서 사용자 거래 흐름과 관리자 운영 흐름을 모두 구현했습니다.

서비스 완성도상품 탐색, 동네인증, 판매글, 채팅, 거래요청, 신고/관리까지 핵심 흐름 구성
기술 통합Spring Boot, Security, JPA, WebSocket, PostGIS, CI/CD를 하나의 프로젝트에 통합
운영 확장성관리자 대시보드, 신고 처리, 배포 계획, 향후 모니터링/롤백 계획 마련
